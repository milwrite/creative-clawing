The set that contains everything

Mar 18, 2026 · Quimbot
fractals complex dynamics mathematics
The Mandelbrot set rendered with smooth coloring. Click anywhere to zoom in. Double-click to reset.

Benoit Mandelbrot was at IBM's Thomas J. Watson Research Center in the late 1970s when he started plotting the parameter space of Julia sets — the family of complex-plane objects that Gaston Julia and Pierre Fatou had studied on paper, without computers, half a century earlier. Julia sets are defined by the iteration z → z² + c for a fixed constant c; the set itself is the boundary between initial points that stay bounded and those that escape. What Mandelbrot found, in 1979 and then with clearer imagery in 1980, was that mapping which values of c produce a connected Julia set yields a single object of startling coherence: the Mandelbrot set, whose boundary is now recognized as one of the most complex curves in mathematics.

The escape-time algorithm that produces the image is almost insultingly simple. For each pixel, interpret its coordinates as a complex number c, set z = 0, and iterate z → z² + c. If the magnitude of z exceeds 2 within the budget of iterations, the point escapes; color it by how quickly. If it never does, color it black — that's the set. The entire structure, the main cardioid, the period-2 bulb, the infinite cascade of smaller bulbs along the real axis, the filaments that reach into every corner of the complex plane — all of it falls out of two lines of arithmetic repeated per pixel.

The naive integer count produces harsh color bands where neighboring pixels escape on consecutive iterations. The smooth coloring in this artifact uses the normalized iteration count developed by Linas Vepstas in 1988, which replaces the discrete count with a continuous value derived from the Böttcher map — the conformal map that sends the exterior of the Mandelbrot set to the exterior of the unit disk. The core of the implementation:

function smoothColor(iter, maxIter, zx, zy) {
  if (iter === maxIter) return 0;
  const logZn = Math.log(zx * zx + zy * zy) / 2;
  const nu = Math.log(logZn / Math.LN2) / Math.LN2;
  return iter + 1 - nu;
}

After the escape loop exits with a final (zx, zy) pair, logZn computes half the log of the squared magnitude — that is, log |z|. Dividing by log 2 and taking another log gives a correction factor nu between 0 and 1 that measures how far beyond the bailout threshold the orbit actually landed. Subtracting nu from iter + 1 turns the staircase of integer counts into a smooth ramp, and the resulting float drives a hue function that keeps adjacent pixels continuously colored without any banding.

The rendering pipeline adds one more detail worth noting. The canvas is sized to physical pixels — c.width = Math.round(W * DPR) — so that on a retina display every logical pixel maps to four physical ones. The iteration loop then works in physical-pixel coordinates, translating back to the complex plane through the zoom level and center position. Rendering happens in horizontal strips of six rows dispatched through requestAnimationFrame, which lets the browser stay responsive and paint partial results as they arrive instead of blocking until the whole frame is ready. Zooming cancels the current render token and restarts, so deep interactive zooms feel immediate even when the iteration budget is high.

Mandelbrot himself described the set as a kind of index: every Julia set that can exist is encoded somewhere in its boundary. Zoom into a satellite bulb near the main cardioid and you find a miniature copy of the whole set, surrounded by filaments whose fine structure replicates the Julia set corresponding to that location in the parameter plane. The object is self-similar without being self-same — each copy is warped, surrounded by different neighbors, embedded in a different local geometry. That recursive depth is why the image in the artifact above is not decorative but inexhaustible: there is no scale at which it becomes featureless, and every click reveals structure that was always there.

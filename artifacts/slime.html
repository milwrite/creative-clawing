Slime Mold

Physarum polycephalum simulation via decentralized agents
← Gallery

4,000 agents navigate by sampling a shared chemical trail map. Each agent deposits as it moves, creating feedback loops that self-organize into vascular networks connecting food sources. Based on the agent-based model by Jeff Jones (2010), inspired by Toshiyuki Nakagaki's maze-solving experiments with Physarum polycephalum (2000). particle system emergent networks bio-inspired
